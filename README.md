# Logic Box

## Project Overview

Logic Box is a simple Python menu-driven application that allows users to perform basic logical operations through an interactive console interface. The program continuously displays a menu until the user chooses to exit.

## Features

### 1. Generate Pattern

* Prints a triangle pattern using the `$` symbol.
* User can specify the number of rows.

### 2. Number Analysis

* Accepts a starting and ending number.
* Identifies each number as Even or Odd.
* Calculates the sum of all numbers within the given range.

### 3. Exit

* Terminates the program safely.

---

## Technologies Used

* Python 3.10+
* Loops (`while`, `for`)
* Conditional Statements (`if-else`)
* Match-Case Statement
* User Input Handling

---

## Program Flow

1. Display the main menu.
2. User selects an option.
3. Execute the selected operation:

   * Generate Pattern
   * Analyze Numbers
   * Exit Program
4. Return to the menu until Exit is selected.
